# Friends React App

## Start project

Clone this repository:

```sh
git clone https://github.com/abazhin/FriendsReactApp.git
cd FriendsReactApp
```

Install dependencies:

```sh
npm install
```

Build the project:

```sh
npm run build
```

Start the project at [`http://localhost:3000`](http://localhost:3000).

```sh
npm start
```
